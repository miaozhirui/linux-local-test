$ 库：jQuery

jQuery 是当今最流行的 JavaScript 类库，为 DOM 操作、事件处理、动画和 ajax 提供了简明易用的接口。

在 SeaJS 里，将与 jQuery 功能类似的类库都称之为 $ 库，比如 YUI2 Core, MooTools Core, 以及 Ext Core 等等。

每个 $ 库都有自己的优缺点和适用场景。在 SeaJS 的模块精选里，我们推荐采用 jQuery, 调用方式如下： 

  module.declare(function(require, exports, module) {
    var $ = require('jquery');
    // 下面，就可以随心所欲调用 jQuery 了。
  });

注意：通过 SeaJS 加载的 jQuery 是一个局部变量了，随便你取什么名字，不会污染全局，甚至可以同时加载多个不同版本。

jQuery 的具体用法，在此就不啰嗦了，强烈推荐： jQuery API

当然，也可以按照具体应用场景，选择自己喜欢的 $ 库。一般来说，只要 稍作修改，都能运行在 SeaJS 的模块环境中。
